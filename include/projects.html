RESPONSIVE LANDING PAGE

This a Landing Page developed from a sketch given. I built this project without using external libraries, just plain HTML and CSS. It's a responsive web site, I used media queries to make it mobile friendly and tablet friendly. Check it out!.

BUDGET APP

This is a functional-based app. This small app, helps people to quickly get an idea of their budget of the current month. I built this app using the concepts of OOP, using IIFE's and a proper structure to be able to join every module with each other. I used objects, arrays, array's methods, functions and of course, DOM manipulation.

THE PIG GAME

This is a game based on a set of rules previously given, where a couple of players can roll the dice and hold the score, update de points to win and unlock the dice after a winner shows up by clicking on the new game button. This project was built using JavaScript functions, booleans to be able to set or change the game state, validations and DOM manipulation.
